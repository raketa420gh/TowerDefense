---

## 1. КОНЦЕПЦИЯ

Игрок — маг, спускающийся в бесконечное подземелье. Главное оружие — **магический посох**, собранный из модульных деталей. Каждый забег уникален благодаря случайным усилениям и сочетанию деталей посоха. Игрок умирает, возвращается в меню, немного прокачивается и спускается снова.

### Уникальная механика
Посох — не просто статат. Это конструктор из слотов, каждый из которых влияет на поведение, визуал и эффекты оружия.

## 2. СТРУКТУРА ПОСОХА

```
[ АРТЕФАКТ ]       ← вершина, активный элемент (меняет тип урона/эффекта)
[ НАВЕРШИЕ ]       ← оправа артефакта (усиливает/модифицирует эффект)
[ РУКОЯТЬ ]        ← хват (влияет на скорость атаки, восстановление маны)
[ ДРЕВКО ]         ← вид дерева (влияет на радиус, пассивные бонусы)
[ НАВЕРШИЕ СНИЗУ ] ← украшение (косметика или небольшой бонус)
```

### Слоты и примеры вариантов

| Слот | Варианты (прототип) | Эффект |
|---|---|---|
| Артефакт | Огненный кристалл, Ледяная сфера, Молния-рун | Тип урона + спецэффект |
| Навершие | Золотое, Серебряное, Обсидиановое | +урон / +скорость / +AOE |
| Рукоять | Кожаная, Деревянная, Металлическая | Cooldown / Mana regen |
| Древко | Дуб, Ясень, Эбен | +радиус / +пробивание / +отдача |
| Нижнее навершие | Стальной шип, Хрустальный шар | Пассивный бонус или косметика |

## 3. РЕАЛИЗОВАННЫЕ МЕХАНИКИ

- **Движение** — виртуальный джойстик, Rigidbody, поддержка бонусов к скорости
- **Посох (автоатака)** — ближайший враг в радиусе детекции, снаряды с cooldown и уроном; пул снарядов
- **Враги** — pathfinding к игроку, object pool, `HealthComponent`, `EnemyConfig` (HP, скорость, XP)
- **Волны** — `EnemySpawner` с `AnimationCurve`; количество и частота спавна нарастают со временем
- **Прогрессия** — XP за убийства, уровни с экспоненциальным скейлингом, выбор 1 из 3 случайных усилений
- **Усиления** — урон %, скорость атаки %, скорость движения %, двойной выстрел, макс. HP, мгновенное лечение
- **HUD** — HP-бар, XP-бар, джойстик
- **Экран поражения** — показывает результаты забега
